## Как собрать APK без компьютера

### Шаг 1. Загрузите проект на GitHub
Проще всего через приложение **GitHub** (или **Working Copy** / **Termux + git**) на телефоне:
1. Создайте новый пустой репозиторий на github.com
2. Загрузите все файлы из этого архива, сохранив структуру папок как есть

### Шаг 2. Сборка запустится автоматически
В репозитории уже лежит `.github/workflows/build.yml`. При каждом push в ветку `main`
GitHub Actions сам:
- поставит JDK и Gradle
- соберёт `app-debug.apk`
- положит его во вкладку **Actions → (последний запуск) → Artifacts**

### Шаг 3. Скачайте APK
Откройте вкладку **Actions** в репозитории на телефоне (через браузер или приложение GitHub),
выберите последний прогон → внизу страницы будет **Artifacts → core-debug-apk** → скачать.
Установите как обычный APK (может понадобиться разрешить "установку из неизвестных источников").

### Если хотите изменить код прямо с телефона
Удобнее всего: приложение **GitHub** → открыть файл → редактировать → Commit.
Каждый коммит в `main` автоматически запускает новую сборку.
